# Админский Комбайн: Cloudflare WARP + Amnezia VPN 🚀

Bash-скрипт для автоматического заворачивания трафика из Docker-контейнеров **Amnezia VPN** (поддерживаются протоколы WG, AWG, AWG2) во внешний туннель **Cloudflare WARP** (через утилиту `wgcf`). 

Позволяет обойти блокировки VPN-протоколов со стороны провайдеров (благодаря Amnezia), сохраняя при этом "чистый" зарубежный IP-адрес Cloudflare на выходе для конечных сайтов.

## 🛠️ Как это работает
1. Скрипт динамически определяет имя запущенного Docker-контейнера Amnezia.
2. Автоматически находит внутренний сетевой bridge-интерфейс Docker и UDP-порт контейнера.
3. Скачивает актуальный релиз `wgcf` с GitHub, регистрирует аккаунт WARP и генерирует WireGuard-конфиг.
4. Создает виртуальный сетевой интерфейс `warp` на хосте.
5. С помощью `iptables` и `fwmark` маркирует пакеты, идущие от контейнера Amnezia, и пускает их по таблице маршрутизации через туннель WARP.

## 📋 Требования
* ОС: Ubuntu / Debian (архитектура x86_64)
* На сервере должен быть предварительно установлен и запущен Docker-контейнер Amnezia VPN.
* Права `root`.

## 🚀 Быстрый запуск

Вы вы можете запустить скрипт напрямую одной командой (убедитесь, что у вас установлен `curl`):

```bash
curl -sSL [https://raw.githubusercontent.com/ВАШ_НИК_НА_GITHUB/ИМЯ_РЕПО/main/mega_kombain.sh](https://raw.githubusercontent.com/ВАШ_НИК_НА_GITHUB/ИМЯ_РЕПО/main/mega_kombain.sh) | sudo bash
